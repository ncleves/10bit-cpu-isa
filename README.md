# 10bit-cpu-isa
A 10 bit Single Cycle CPU written primarily in Verilog that solves two "simple" problems. The instruction set architecture was created for this CPU as well.

```
| Register|Address|
|---|---|
|  r0 |    000|
|  r1 |    001|
|  r2 |    010|
|  r3 |    011|
|  r4 |    100|
|  r5 |    101|
|  r6 |    110|
|  r7 |    111|
```
